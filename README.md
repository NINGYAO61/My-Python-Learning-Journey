情绪日记（Emotion Diary）
一个轻量型命令行情绪记录工具，支持情绪记录、历史查询和个性化情绪建议，帮助你追踪情绪变化、释放内心状态。
项目介绍
这是一个基于 Python 开发的命令行工具，无需复杂配置，开箱即用。通过简单的数字选择操作，即可快速记录每日情绪、回顾历史记录，还能根据当前情绪获取针对性小建议，适合日常情绪管理使用。
核心功能
记录今日情绪：输入日期、选择情绪类型（开心 / 难过 / 平静 / 烦躁），可添加备注补充细节
查看历史记录：按记录顺序展示所有保存的情绪数据，包含日期、情绪和备注
获取情绪建议：根据当前情绪提供简单易操作的调节建议
安全退出：支持随时退出程序，操作流程简洁直观
运行步骤
环境要求：确保本地安装 Python 3.6 及以上版本
下载代码：将情绪日记脚本保存为 emotion_diary.py
运行程序：
打开命令行终端，切换到脚本所在目录
执行命令：python emotion_diary.py
按照终端提示输入数字（1-4）选择对应功能

退出模块：打印告别提示并终止程序

注意事项
日期输入建议遵循格式 YYYY-MM-DD（例如：2025-10-26），便于后续查看时的可读性
情绪选择需输入数字 1-4，输入其他数字会标记为「未明确」情绪
数据仅在程序运行期间临时存储，退出程序后记录会丢失（需持久化可扩展文件存储功能）
备注为可选输入，不填写时会显示空值，不影响记录保存
